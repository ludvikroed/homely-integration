# Homely API Documentation

This folder contains the official Homely API document and example data structures.

- `Homely_API_documentation.md`
  The official documentation received from Homely. Read this first.

- `data_structure/`
  Example data structures based on real API data and the official Homely examples.

## Notes

- The files in `data_structure/` are meant to help show how the JSON is structured.
- The official document is still the main reference for endpoint behavior and overall API usage.
- The best way to read this folder is to use both the official document for the API description and the data structure files for concrete examples.
